Installation
=============


1)
Copy the files from the 'magento' directory to your target 
installation directory of Magento.

2) Login to admin go to System -> configuration -> Kodecrm -> Settings. Set the 'Feed Status Enabled' to 'Yes'.
Your feed url will be http://yourdomain.com/index.php/kodecrm.

3) Go to System -> Configuration -> General -> Design -> Footer and add the chat snippet in 'Miscellaneous HTML' field.
The chat snippet can be taken from Kodecrm admin setting page.
